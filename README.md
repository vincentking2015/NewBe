# NewBe

一个超级简单的PHP的MVC框架。

学习框架，就是为了学习设计思想，本项目是一个非常简单的PHP的MVC框架，麻雀虽小，但是五脏俱全。把很多功能弱化，方便学习，也会高效

结合tp的基本目录结构，基本实现思路也很像

本框架是整合   
    Twig视图模板引擎
    MeDoo的数据库框架

唯一入口模式
    public文件夹入口
    applicatin项目目录
    resource的view是视图文件
    config是配置文件
    newbe文件夹是core代码


安装
直接下载，在composer.json文件夹使用composer install
